SourceFence

Sourcing Compliance, Simplified.

SourceFence is a Chrome extension that helps recruiters maintain sourcing compliance on LinkedIn. Set up restriction rules for locations and companies, and SourceFence flags matching candidates in real time as you browse profiles and search results.

Location Rules Flag candidates from restricted regions, countries, or cities
Company Rules Enforce non-solicit agreements and off-limits clients
Real-Time Alerts Banners on profiles and badges on search results
Privacy First All data stays in your browser. Zero external requests.

Where it works

  • LinkedIn standard profiles (/in/)
  • LinkedIn Recruiter (/talent/)
  • LinkedIn search results with per-candidate badges
